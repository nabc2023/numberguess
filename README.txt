# Number Guesser

This is a simple number guessing game. The program generates a random number between 1 and 10, and the user has to guess what the number is.

## Usage

To use this app, simply run the script. The program will generate a random number between 1 and 10, and then ask you to guess what the number is. If you guess the number correctly, you win! If not, the program will tell you if your guess was too high or too low, and then ask you to guess again.
